Fontlog for Fälschungsershwerende Schrift

This file provides detailed information on the Fälschungsershwerende Schrift font software. This information should be distributed along with the Fälschungsershwerende Schrift font and any derivative works.

Basic Font Information

Fälschungsershwerende Schrift is a font in one weight originally drawn by Karlgeorg Hoefer in 1978. This version is based on a specimen that was published with the essay “Fälschungsershwerende Schrift” by Benjamin Tiven (http://www.servinglibrary.org/journal/3/falschungserschwerende-schrift).

Fälschungsershwerende Schrift is a Unicode typeface family that supports all languages that use the Latin script and its variants, and could be expanded to support other scripts.
